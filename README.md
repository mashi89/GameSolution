# GameSolution

## Wood cutting game

### Core game loop
1. Enter forest
2. Cut trees with axe
3. Take trees to sled
4. Day ends on timer --> Count up money from trees depending on size and quality
5. Sleep
6. Repeat

### Additional game elements
#### Stamina
- Expend stamina by using axe
- Regain stamina by consuming restoratives

#### Items
- Inventory management
  - Items
  - Restoratives
  - Equipments
- Equipment progression
  - Get better axes
  - Get better clothes
  - Get better sled
